# New Starter IT Onboarding Checklist

## Before the Start Date

- Confirm the starter's full name, department, job role, manager, and start date
- Create the user account
- Assign required licences
- Add the user to approved groups
- Prepare and label the device
- Install required software
- Apply updates and security settings
- Test sign-in and network access

## On the First Day

- Confirm the user can sign in
- Set up Microsoft 365 applications
- Configure Outlook, Teams, and OneDrive
- Support MFA registration
- Explain password and security expectations
- Introduce key business or school applications
- Show the user how to request IT support

## After Onboarding

- Confirm the user has access to everything needed
- Resolve missing permissions
- Update asset records
- Record completion in the ticketing system
- Follow up if additional training is needed

## Goal

A successful onboarding should help the user feel confident, supported, and productive from the start.
