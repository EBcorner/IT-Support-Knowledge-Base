# Active Directory User Account Troubleshooting

## Common Requests

- Password reset
- Locked account
- Missing group access
- Incorrect permissions
- New starter account setup
- Leaver account disablement

## Troubleshooting Process

1. Confirm the user's identity using the approved verification process.
2. Locate the account in Active Directory.
3. Check whether the account is enabled or locked.
4. Review password status and expiry.
5. Confirm group memberships.
6. Check the correct organisational unit.
7. Compare access with another user in a similar role where appropriate.
8. Make only authorised changes.
9. Ask the user to test access.
10. Document all changes in the ticket.

## Security Reminder

Never share passwords in plain text or make access changes without confirming authorisation.
