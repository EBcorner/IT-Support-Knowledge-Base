# OneDrive Sync Troubleshooting

## Common Symptoms

- Files are not syncing
- A red X appears beside files
- OneDrive is signed out
- Changes are not appearing on another device
- The OneDrive client is stuck on processing changes

## Troubleshooting Steps

1. Check internet connectivity.
2. Confirm the user is signed into the correct work account.
3. Review the OneDrive icon for warnings or sync errors.
4. Check available storage space locally and in Microsoft 365.
5. Confirm file names do not contain unsupported characters.
6. Pause and resume syncing.
7. Restart OneDrive.
8. Sign out and sign back in if required.
9. Check Microsoft 365 service health for a wider incident.
10. Record the resolution in the support ticket.

## Good Support Practice

Explain each step clearly and confirm that the user's files are safe before making changes to the account or sync configuration.
