# macOS Basic Support Checklist

## Common Areas

- User sign-in
- Wi-Fi connectivity
- Software installation
- Storage space
- Application permissions
- Audio, camera, and microphone
- External displays
- System updates

## Basic Checks

1. Confirm the macOS version.
2. Restart the affected application.
3. Check System Settings for permissions.
4. Confirm sufficient disk space.
5. Review network connection.
6. Test with another user profile where appropriate.
7. Check for updates.
8. Restart the Mac if safe to do so.
9. Capture error messages and logs before escalation.

## Key Lesson

macOS support often depends on checking privacy permissions, account access, system updates, and hardware connections.
