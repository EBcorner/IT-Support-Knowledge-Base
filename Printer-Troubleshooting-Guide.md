# Printer Troubleshooting Guide

## Common Issues

- Printer showing offline
- Jobs stuck in the print queue
- Poor print quality
- Paper jam
- Incorrect tray or paper size
- User unable to find the printer

## Troubleshooting Steps

1. Confirm power, network, and cable connections.
2. Check the printer display for warnings.
3. Confirm the correct printer is selected.
4. Clear stuck print jobs.
5. Restart the print spooler if required.
6. Print a test page.
7. Check toner, paper, and consumables.
8. Confirm the IP address or print server connection.
9. Reinstall or reconnect the printer if authorised.
10. Document the fault and resolution.

## Support Tip

Always confirm whether the problem affects one user or multiple users before changing the printer configuration.
